### 数据来源
- 股票行情：akshare（`ak.stock_zh_a_hist()`），后复权，日度
- 市场指数：沪深300（000300，必选，CAPM市场基准）；中证500（000905，自选，代表中小盘风格，与沪深300大盘风格形成互补）
- 宏观指标：
  - CPI同比增速，来源：akshare（`ak.macro_china_cpi_yearly()`），必选指标，反映通胀水平，直接影响货币政策和市场估值
  - M2同比增速，来源：akshare（`ak.macro_china_money_supply()`），选择理由：M2增速反映货币政策宽松程度，与股市流动性密切相关，是预判市场走势的重要先行指标
- 财务数据：akshare（`ak.stock_financial_analysis_indicator()`），获取ROE和净利润率

### 存储方式
- 基础：CSV（方式 A）—— 通用性好，易于阅读和交换，是数据处理的标准格式
- 进阶：SQLite（方式 C）—— 选择理由：SQLite是轻量级关系数据库，无需额外服务进程，适合中小规模金融数据管理；支持SQL查询和跨表JOIN，便于复杂的数据关联分析；单文件存储方便管理
- CSV优点：通用性极强，所有工具都支持；纯文本可读；简单直观。力不从心的场景：数据量大时读写速度慢；不支持列式读取，每次需加载全部数据；缺乏类型约束和数据校验；并发写入困难

### 如何运行
1. 安装依赖：`pip install -r requirements.txt`
2. 运行 `01_download.ipynb` 下载原始数据
3. 运行 `02_clean.ipynb` 清洗并存储数据
4. 运行 `03_analysis.ipynb` 查看分析结果
5. 打开 `report.html` 阅读完整报告

### 数据说明
- `data/clean/` 中的清洗后数据已上传，可直接使用
- `data/combined/` 中的合并后数据已上传（不含 .db 文件）
- 原始数据未上传（体积较大），需运行 `01_download.ipynb` 重新下载
- 如需重建 SQLite 数据库，运行 `02_clean.ipynb` 中相关单元格即可
